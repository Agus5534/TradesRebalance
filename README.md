# TradesRebalance
A plugin that replicates [Minecraft 23w31a](https://www.minecraft.net/en-us/article/minecraft-snapshot-23w31a) librarian villagers nerf.

Valid from Minecraft 1.17 to latest release.

Uses Paper software.
